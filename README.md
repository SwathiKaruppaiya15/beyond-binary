# beyond-binary
Unique collection of DSA solutions &amp; problem-solving journey

## Structure
- `Arrays/`, `Strings/`, `LinkedList/`, `Trees/`, `DynamicProgramming/`
- Each topic has `Easy/`, `Medium/`, `Hard/` folders (when applicable).

## How to contribute
- Add solutions into the appropriate folder.
- Filename format: `<ProblemNumber>_<Problem-Title>.<ext>` e.g. `1_Two_Sum.py`
- Commit message format: `Add: <Problem Title> (Difficulty, Language)`
- Add a short comment at top of file: problem link + complexity.
